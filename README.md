# Avia - Assistant Intelligent de Gestion de Rendez-vous

## Description
Avia est un assistant intelligent basé sur l’IA (Gemini) qui permet de gérer une base de données SQLite via langage naturel.

---

## Lancement de l'application

### 1. Backend
Double-clique sur :

start_backend.bat

Le serveur sera accessible sur :
http://127.0.0.1:8000

---

### 2. Frontend
Double-clique sur :

start_frontend.bat

---

## Fonctionnalités

- Dialogue en langage naturel
- Gestion de base de données SQLite
- Génération automatique de requêtes SQL
- Interprétation intelligente des résultats
- Mémoire conversationnelle
- Assistant IA personnalisé

---

## Technologies utilisées

- Python (FastAPI)
- Google Gemini AI
- SQLite
- Flutter
- Requests
- Dotenv
